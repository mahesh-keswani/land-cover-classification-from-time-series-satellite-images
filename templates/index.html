In the past, scientists had to physically travel to places to do Land Cover Classification. But due to advancements in satellite imagery, this process can be simplified. In this project, we tried to do land cover classification using Landsat-8 Satellite Data.

Features

Our Project has the following features:

  • To detect change in land cover
  • To pixelwise classify the landcover
  • Map
  • Upload TIFF Image File
  • Example places given
